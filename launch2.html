Pour démarrer l'appli en local : 

1. Démarrer Wampserver64 pour Apache/localhost, MySQL, PhpMyAdmin :
    ⊞ wa Enter OK
  et attendre une minute ou deux que tout se lance
2. Connecter à la base pour vérifier que A, M, P sont bien lancés :
    http://localhost/phpmyadmin/index.php?route=/&route=%2F&db=jdb1
  avec login=root / password vide
3. Lancer le serveur web local du framework Symfony :
  ouvrir un terminal Windows PowerShell, on se trouve dans C:\Users\kaica\
  aller dans le répertoire où se trouvent les sources de l'appli Jourdebord :
    cd lgc\Jdb1
  lancer le serveur web local du framework Symfony
    symfony serve
4. Ouvrir la page d'accueil de Jourdebord dans un browser :
    http://127.0.0.1:8000/
